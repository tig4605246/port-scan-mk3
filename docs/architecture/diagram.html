port-scan-mk3 Architecture (Static Block Flow)

CLI orchestration, core package flow, and e2e artifact validation path.

CLI Layer

CLI

port-scan validate

port-scan scan

config

Parse flags and normalize runtime options.

CLI → input

Core Package Layer

input

Load CIDR/port files and run fail-fast validation.

task

Expand selectors and create scan tasks.

scanapp

Orchestrate dispatch, workers, progress, and completion.

writer

Write batch CSV outputs (full + open-only).

speedctrl

Pause gate driven by manual control and pressure API.

state

Persist and reload resume state (resume_state.json).

logx

Structured runtime events and diagnostics.

pressure API → speedctrl → scanapp
scanapp interruption/failure → state (resume_state)
scanapp results → writer → e2e/out artifacts
